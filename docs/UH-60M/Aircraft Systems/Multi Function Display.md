# Multi Function Display (MFD)

Explains the various pages displayed on the screens in front of the pilot and co-pilot

There are 6 pages that can be displatyed on the MFDs via the bezel keys:
1. Primary Flight Display (PFD)

2. Primary Flight Display(PFD)

3. Engine Instrument Caution/Advisory System(EICAS)

4. Navigation Display(ND)

4. TAC Map(TAC)

5. Joint Variable Message Format(JVMF)

6. Integrated Vehicle Health Management System(IVHMS) - *Coming soon*
